Word 67 / adjective

Dormant

DOR-muhnt · UK /ˈdɔː.mənt/ · US /ˈdɔːr.mənt/

不是不存在，而是暫時不活躍、沒有表面動作，但狀態與可能性還在。

今日定位

把 dormant 記成「inactive but still there」：不是消失，只是暫時沒醒。

Core Idea

dormant 用來描述火山、帳號、功能、情緒或問題目前不活躍，但並沒有真正消失。

概念焦點

核心是潛伏存在，而不是結束。

語氣質地

常見於地質、技術、風險、商務與心理狀態描述。

使用提醒

不要把 dormant 當成 gone；它常暗示後續還可能再次被喚起。

Meaning

簡短定義

Dormant

形容火山、帳號、功能、情緒或問題目前不活躍，但並沒有真正消失。

不是 Inactive

inactive 只是沒在動；dormant 更像睡著，之後還可能醒來。

曾經存在或可啟動
→
暫時沉下去
→
必要時可能再起來

Origin

字源來源

dormant 和拉丁語裡「睡覺」有關，所以它天然帶有休眠而非消滅的感覺。

字源畫面可以記成：火沒有熄掉，只是暫時埋在灰下面。

Memory Hook

記憶鉤子

dormant 是看起來安靜，但你知道它還在。

很多風險最難處理的地方，就是它 dormant 到讓人誤以為不存在。

Usage

生活與專業情境

日常版

有些念頭平常是 dormant 的，但一被觸發就又浮上來。

數位 / 專業版

在系統裡，a dormant bug 可能很久沒被撞到，但一換環境就再次出現。

延伸版

地質、法律、資安與產品功能切換都常用 dormant 來描述休眠狀態。

Collocations

常見搭配與語域

Dormant 常和 volcano、account、gene、bug、feature 搭配。

a dormant volcano 自然 / 地質

表示目前不噴發，但仍被視為活體風險。

a dormant account 商務 / 系統

表示帳號存在但長期未活躍。

a dormant issue 工程 / 風險

表示問題暫時沉下去，仍可能回來。

Neighbors

對立與鄰近概念

概念 核心精神 最自然的使用時機
Dormant 暫時休眠，但仍在場 談風險、功能、帳號、情緒或火山時
Inactive 只是目前沒有活動 不特別暗示未來可能被喚醒時

Modern Use

現代延伸

現代英文裡，dormant 很常被拿來描述暫時不活躍但仍具潛在影響的事物，從火山到 bug 都適用。

它的價值在於提醒你：表面安靜不代表系統裡已經沒有那個東西。
