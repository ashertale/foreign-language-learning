Word 64 / adjective

Cumulative

KYOO-myuh-luh-tiv · UK /ˈkjuː.mjə.lə.tɪv/ · US /ˈkjuː.mjə.lə.tɪv/

不是只是一再重複，而是每一次都往同一個總量上再加一層。

今日定位

把 cumulative 記成「it keeps adding up」：重點不是單次大，而是累積後變大。

Core Idea

cumulative 用來描述影響、資料、成本、疲勞或效果會一層一層疊上去，而不是每次重來。

概念焦點

核心是疊加總量，不是單次事件。

語氣質地

常見於醫療、教育、產品、法律、風險與數據分析。

使用提醒

不要把 cumulative 當成 repeated；重複不一定累加，但 cumulative 一定是在同一筆帳上越堆越多。

Meaning

簡短定義

Cumulative

不是只是一再重複，而是每一次都往同一個總量上再加一層。

不是 Repeated

repeated 只說一再發生；cumulative 更強調每次發生都留下總量效果。

先有一次影響
→
後面繼續疊上
→
總效果變大

Origin

字源來源

cumulative 和拉丁語裡的「堆起來、堆積」有關，所以它的畫面一直都很具體：不是散掉，而是往上加。

字源畫面可以記成：你不是一直擦掉重算，而是每次都把新的一層放上去。

Memory Hook

記憶鉤子

cumulative 是單次看不嚇人，總和回頭一看才嚇人。

很多負擔不是因為一次太大，而是小量長期疊加後變成整體重量。

Usage

生活與專業情境

日常版

睡眠不足的傷害常是 cumulative，一天沒感覺，連幾天就會很明顯。

數位 / 專業版

在系統設計裡，small inefficiencies can have a cumulative impact on latency and maintenance cost.

延伸版

醫療、財務、教育與風險管理都常用 cumulative 來談累積效應。

Collocations

常見搭配與語域

Cumulative 常和 effect、impact、cost、risk、record 搭配。

cumulative effect 一般 / 分析

表示影響會一層層疊加。

cumulative cost 財務 / 專案

表示總成本是逐步堆上來的。

cumulative risk 風險 / 醫療

表示風險不是單次，而是累積後升高。

Neighbors

對立與鄰近概念

概念 核心精神 最自然的使用時機
Cumulative 每次都把總量再往上堆一點 談效果、成本、疲勞、風險或數據總量時
Repeated 一再發生，但不一定留下總量 重點只是頻率，而不是堆加效果時

Modern Use

現代延伸

現代英文裡，cumulative 幾乎是談長期效應時的核心字，因為很多問題都不是單次衝擊，而是疊加後出現。

在工程與產品語境中，它很適合描述小延遲、小 workaround 或小負債最後一起拖垮系統的方式。
