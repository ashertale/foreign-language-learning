Word 82 / noun

Inflection

in-FLEK-shuhn · UK /ɪnˈflek.ʃən/ · US /ɪnˈflek.ʃən/

不是隨便的變化，而是聲音、語意、曲線或局勢在某個點開始轉彎。

今日定位

把 inflection 記成「the bend or turn that changes the pattern」：不是所有變化都算，得是那個轉折點。

Core Idea

inflection 用來描述語音抑揚、語法變化，或更抽象的趨勢與局勢出現轉向。

概念焦點

核心是轉折與彎曲，不只是差異。

語氣質地

常見於語言學、聲音、數據曲線與市場敘事。

使用提醒

不要把 inflection 當成一般 tone shift；它更像模式在某點上開始轉向。

Meaning

簡短定義

Inflection

不是隨便的變化，而是聲音、語意、曲線或局勢在某個點開始轉彎。

不是 Tone

tone 可以只是語氣；inflection 則更像聲音或曲線上的轉折點。

先有原本走向
→

→

Origin

字源來源

inflection 和拉丁語裡「彎曲、轉折」有關，所以它很自然地適合描述曲線、語氣和局勢轉向。

字源畫面可以記成：不是整條線消失，而是在某一點開始轉彎。

Memory Hook

記憶鉤子

inflection 是你聽到或看到走勢開始改變的那個彎點。

真正重要的變化常不是數字本身，而是 inflection point 出現在哪裡。

Usage

生活與專業情境

日常版

一句話的最後一點點 inflection，就可能讓聽起來像提問或像質疑。

數位 / 專業版

在 metrics review 裡，大家常盯的是 trend 的 inflection，因為那裡往往代表系統行為變了。

延伸版

語言學、數據科學、商業分析與公開表達都常用 inflection 來談轉折。

Collocations

常見搭配與語域

Inflection 常和 point、voice、curve、trend 搭配。

inflection point 商務 / 分析

表示走勢開始改變的關鍵點。

vocal inflection 語言 / 表達

表示聲音高低轉折的細節。

inflection in the curve 數據 / 技術

表示曲線形狀出現轉向。

Neighbors

對立與鄰近概念

概念 核心精神 最自然的使用時機
Inflection 模式在某點上開始轉彎 談語氣、曲線、趨勢或局勢轉向時
Tone 一般語氣或聲調 只想說口氣，不強調轉折點時

Modern Use

現代延伸

現代英文裡，inflection 一邊保留語言學意義，一邊廣泛被借去描述趨勢轉折。

它很適合提醒人：真正重要的不是一直線上的數字，而是曲線何時開始彎。
