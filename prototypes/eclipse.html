Eclipse

ih-KLIPS · UK /ɪˈklɪps/ · US /ɪˈklɪps/

不是只是比它好，而是好到它幾乎看不見了。

今日定位

Core Idea

eclipse 表示遮蔽、掩蓋，或因更強勢的事物而使別的東西失色。

概念焦點

核心畫面是 block or outshine something so it loses visibility。

語氣質地

這個字常見於 競品比較、天文比喻與敘事焦點；語氣上不是空泛形容，而是用來抓住 block or outshine something so it loses visibility 這種狀態或動作。

使用提醒

不要把 eclipse 只看成 outshine；outshine 多半只是 蓋過，而 eclipse 更強調 block or outshine something so it loses visibility。
