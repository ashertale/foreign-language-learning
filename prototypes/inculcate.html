Inculcate

IN-kul-kayt · UK /ˈɪn.kʌl.keɪt/ · US /ˈɪn.kʌl.keɪt/

不是說一次，而是反覆灌溉到它變成習慣性反應。

Core Idea

用 inculcate 時，句子通常在做一個收焦動作：把讀者帶到「反覆灌溉到它變成習慣性反應」那個更窄的位置。

概念焦點

如果上下文沒有讓「反覆灌溉到它變成習慣性反應」變重要，這個字就不必上場。

語氣質地

它的語氣像把句子往內收一格：少一點泛稱，多一點可判斷的輪廓。

使用提醒

使用時要避開翻譯腔：先有情境，再讓 inculcate 進來，不要先決定一定要用它。

Meaning

簡短定義

Inculcate

表示反覆灌輸觀念、價值或習慣。

不是 Instill

如果 instill 已經能讓句子成立，就不必硬換；需要「反覆灌輸觀念、價值或習慣」時，inculcate 才有必要。

Usage

生活與專業情境

日常版

inculcate discipline 是最低摩擦入口；它讓你不用先背完整定義，也能感到「反覆灌溉到它變成習慣性反應」。

數位 / 專業版

inculcate the habit 適合拿來檢查語域：它讓 inculcate 從認得走向會判斷。

延伸版

用 inculcate values 造句時，不要寫得花；只要讓讀者清楚看見「反覆灌輸觀念、價值或習慣」。

Collocations

常見搭配與語域

把搭配當成三個入口：inculcate discipline 給基本場景，inculcate the habit 看語域，inculcate values 測你能不能輸出。

inculcate discipline 教育 / 成長

inculcate discipline 是入口，不是答案；用它開始，再往上下文看。

inculcate the habit 學習 / 生活

讀 inculcate the habit 時，重點是語境，不是把搭配背成清單。

inculcate values 組織 / 文化

inculcate values 用來測試輸出；能自然使用它，才算真的碰到邊界。

Modern Use

現代延伸

實際閱讀時，inculcate 多半不會孤立出現。先留意 inculcate discipline、inculcate the habit、inculcate values 這類搭配如何替它定形。

如果一句話只需要大意，別急著用 inculcate；如果要處理「反覆灌溉到它變成習慣性反應」，它才值得出場。
