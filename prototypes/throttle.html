Throttle

THRAHT-l · UK /ˈθrɒt.əl/ · US /ˈθrɑː.t̬əl/

不是完全關掉，而是把流量壓小，讓系統別一下子衝太快。

Core Idea

用 throttle 時，句子通常在做一個收焦動作：把讀者帶到「把流量壓小，讓系統別一下子衝太快」那個更窄的位置。

概念焦點

先看搭配，再看定義。throttle requests、throttle the engine、throttle network traffic 會告訴你這個字實際站在哪裡。

語氣質地

放進 系統 / API 時它比較容易入口；到了 機械 / 車輛 或 網路 / 維運，你會看到同一個核心如何換場景。

使用提醒

如果你無法說明「把流量壓小，讓系統別一下子衝太快」在哪裡，這個字多半只是被塞進句子。

Meaning

簡短定義

Throttle

表示節流、限制速度、頻寬或資源輸出。

不是 Limit

limit 可以先說出 節制；throttle 則把句子推到「把流量壓小，讓系統別一下子衝太快」這個更細的位置。
