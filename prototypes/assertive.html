Assertive

uh-SUR-tiv · UK /əˈsɜː.tɪv/ · US /əˈsɝː.t̬ɪv/

不是兇，而是把界線與立場說清楚，不靠退縮也不靠壓人。

今日定位

Core Idea

用 assertive 時，句子通常在做一個收焦動作：把讀者帶到「把界線與立場說清楚，不靠退縮也不靠壓人」那個更窄的位置。

概念焦點

如果上下文沒有讓「把界線與立場說清楚，不靠退縮也不靠壓人」變重要，這個字就不必上場。

語氣質地

它的語氣像把句子往內收一格：少一點泛稱，多一點可判斷的輪廓。

使用提醒

使用時要避開翻譯腔：先有情境，再讓 assertive 進來，不要先決定一定要用它。

Meaning

簡短定義

Assertive

表示表達主張時清楚、直接、有邊界感。

不是 Aggressive

aggressive 可以先說出 咄咄逼人或高壓；assertive 則把句子推到「把界線與立場說清楚，不靠退縮也不靠壓人」這個更細的位置。

從 溝通 / 人際 的例子入手
→

→
最後用 assertive response 測試輸出

Origin

字源來源

不要把來源當背誦題；把它當成一條輔助線，幫你從「像門是穩穩關上的，不是砰一聲甩上去」回到「把界線與立場說清楚，不靠退縮也不靠壓人」。

Usage

生活與專業情境

日常版

先拿 assertive tone 寫一個你真的會遇到的場景；這比多看三個中文解釋有用。

數位 / 專業版

讀 assertive leadership 時，注意它前後通常會提供理由，讓「把界線與立場說清楚，不靠退縮也不靠壓人」變得必要。

延伸版

assertive response 是延伸練習。把它放進句子後，確認意思仍回到「把界線與立場說清楚，不靠退縮也不靠壓人」。

Collocations

常見搭配與語域

把搭配當成三個入口：assertive tone 給基本場景，assertive leadership 看語域，assertive response 測你能不能輸出。

assertive tone 溝通 / 人際

assertive tone 先建立基本語境；它讓定義有地方落腳。

assertive leadership 管理 / 團隊

用 assertive leadership 練習時，先看它前後是否支持「把界線與立場說清楚，不靠退縮也不靠壓人」。

assertive response 工作 / 協作

把 assertive response 寫進自己的句子，檢查語氣是否仍回到「把界線與立場說清楚，不靠退縮也不靠壓人」。

Modern Use

現代延伸

遇到 assertive 時，先看旁邊的名詞或動詞。它們會告訴你是 溝通 / 人際、管理 / 團隊，還是 工作 / 協作 的用法。

真正掌握它的標準是：你能換一個情境，仍然說清楚為什麼這裡該用 assertive。
