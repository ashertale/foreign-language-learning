Word 488 / adjective

Onerous

ON-er-us · UK /ˈəʊ.nər.əs/ · US /ˈɑː.nɚ.əs/

不是麻煩而已，而是重到你做每一步都覺得拖。

今日定位

先用 onerous requirement 和 onerous workload 把語感站穩，再回頭看定義；順序反過來時，這個字很容易變成空殼。

Core Idea

onerous 最值得保留的語感，是它讓「繁重、負擔很大的」不再只是模糊大意。

概念焦點

焦點不是背出長定義，而是知道什麼時候「繁重、負擔很大的」真的在句子裡發生。

語氣質地

語氣落在 制度 / 合規、工作 / 管理、法務 / 合約 之間；它不一定華麗，但通常比日常大詞更願意把邊界說清楚。

使用提醒

不要把 onerous 用成 burdensome 的自動升級。若句子只需要 負擔沉重，較寬的字反而更自然。

Meaning

簡短定義

Onerous

表示繁重、負擔很大的。

不是 Burdensome

兩者不是難易差，而是焦距差。burdensome 像外框，onerous 像把鏡頭對到關鍵細節。

Origin

字源來源

字源可以幫忙，但這頁不把 onerous 硬講成故事。先把現代用法穩住：它會把句子帶向「重到你做每一步都覺得拖」。

這個鉤子只負責喚回語感：像肩上不是掛一件外套，而是背著濕透的毯子一路走

Memory Hook

記憶鉤子

onerous：像肩上不是掛一件外套，而是背著濕透的毯子一路走

記憶鉤子不是答案，它只是把你帶回搭配：onerous requirement、onerous workload、onerous obligation。

Usage

生活與專業情境

日常版

用 onerous requirement 練第一句。先讓句子自然，再想它為什麼需要 onerous。

數位 / 專業版

如果要在工作或分析文本裡用，先從 onerous workload 看語氣；它會告訴你這個字能不能承受正式語境。

延伸版

在 法務 / 合約 裡，onerous obligation 讓你看到這個字不是只屬於第一個例子。

Collocations

常見搭配與語域

搭配不是附錄，而是用法本體；onerous 通常是在這些語塊裡變得清楚。

onerous requirement 制度 / 合規

讀 onerous requirement 時，注意搭配對象如何逼出「重到你做每一步都覺得拖」。

onerous workload 工作 / 管理

onerous workload 讓你看到這個字如何進入較正式或較專門的語境。

onerous obligation 法務 / 合約

onerous obligation 不用死背；拿它確認你能不能活用 onerous。

Neighbors

對立與鄰近概念

概念 核心精神 最自然的使用時機
Onerous 繁重、負擔很大的 用在你想讓讀者看到「繁重、負擔很大的」這個較窄邊界的時候。
Burdensome 負擔沉重 用在大方向比細邊界重要的句子；它讓意思先站住，不急著精修。

Modern Use

現代延伸

把它當成閱讀中的訊號：作者正在要求你比一般說法多看一層「重到你做每一步都覺得拖」。

和 burdensome 相比，onerous 的價值不是更難，而是更能說明「繁重、負擔很大的」。
