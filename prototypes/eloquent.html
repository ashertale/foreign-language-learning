Word 69 / adjective

Eloquent

EL-uh-kwuhnt · UK /ˈel.ə.kwənt/ · US /ˈel.ə.kwənt/

不是只會說，而是能把想法講得有力量、有節奏，而且讓人真的聽進去。

今日定位

把 eloquent 記成「speech with force and shape」：不是話多，而是話有穿透力。

Core Idea

eloquent 用來描述語言、文字，甚至沉默或姿態能夠有力而優雅地表達意思。

概念焦點

核心不是流利，而是表達有力量也有形狀。

語氣質地

常見於寫作、演說、評論與人物描寫。

使用提醒

不要把 eloquent 當成 fluent；流利不一定有思想與力度，eloquent 通常兩者兼具。

Meaning

簡短定義

Eloquent

不是只會說，而是能把想法講得有力量、有節奏，而且讓人真的聽進去。

不是 Fluent

fluent 強調說得順；eloquent 更像說得有力、有美感，而且能打中人。

Origin

字源來源

eloquent 和拉丁語裡「說出來」有關，但它後來發展出的重點是說得有分量，而不只是開口。

字源畫面可以記成：不是把詞丟出去，而是讓每一句都帶著方向與重量落地。

Memory Hook

記憶鉤子

eloquent 是你不只聽懂，還感覺到話在心裡留下形狀。

真正 eloquent 的表達，不一定很長；有時一句話就夠把整個場面定住。

Usage

生活與專業情境

日常版

有些人講故事很 eloquent，你會覺得畫面和情緒都被他帶出來了。

數位 / 專業版

在提案或 review 裡，an eloquent explanation 不是術語多，而是複雜事被講得清楚又有說服力。

延伸版

演說、寫作、文學評論與人物刻畫都常用 eloquent 來形容表達的力量。

Collocations

常見搭配與語域

Eloquent 常和 speech、argument、prose、silence 搭配。

an eloquent speech 演說 / 公開表達

表示發言有結構也有感染力。

eloquent prose 寫作 / 文學

表示文字不只清楚，還有美感與力度。

an eloquent silence 描寫 / 修辭

表示連沉默都在表達意思。

Neighbors

對立與鄰近概念

概念 核心精神 最自然的使用時機
Eloquent 把意思說得有力量也有形狀 談演說、寫作、論證或人物表達力時
Fluent 說得順、不卡頓 重點只是流暢，不涉及表達張力時

Modern Use

現代延伸

現代英文裡，eloquent 仍是評價高品質表達的好字，因為它兼顧清楚、力度與美感。

它也常被借去形容非語言訊號，像是表情、姿態或沉默本身就很會說話。
