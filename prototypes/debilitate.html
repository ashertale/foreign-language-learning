Debilitate

dih-BIL-ih-tayt · UK /dɪˈbɪl.ɪ.teɪt/ · US /dɪˈbɪl.ə.teɪt/

不是累而已，而是弱到原本能做的事開始做不動。

今日定位

把 debilitate 記成「weaken until normal function suffers」：不是累而已，而是弱到原本能做的事開始做不動。

Core Idea

debilitate 表示使虛弱、削弱功能或行動力。

概念焦點

核心畫面是 weaken until normal function suffers。

語氣質地

這個字常見於 健康狀態、系統損耗與心理壓力；語氣上不是空泛形容，而是用來抓住 weaken until normal function suffers 這種狀態或動作。

使用提醒

不要把 debilitate 只看成 weaken；weaken 多半只是 削弱，而 debilitate 更強調 weaken until normal function suffers。

Origin

字源來源

學 debilitate 時，先不用把它當成冷知識題；更重要的是抓住它如今穩定的核心畫面：weaken until normal function suffers。

把歷史感先壓縮成一個可記的畫面：像電池不是掉一格，而是掉到連開機都變得吃力。

Memory Hook

記憶鉤子

debilitate 的畫面是：像電池不是掉一格，而是掉到連開機都變得吃力。

只要你記得這個畫面，下次看到 debilitate，就不會只把它翻成單薄的中文對應。

Usage

數位 / 專業版

延伸版
