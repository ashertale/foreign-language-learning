Word 13 / adjective

Cogent

KOH-jent · UK /ˈkəʊ.dʒənt/ · US /ˈkoʊ.dʒənt/

不是單純清楚，而是清楚到能把理由推到一起，讓人覺得站得住腳。

今日定位

用在論證、說明、分析、提案與 code review 理由；它稱讚一段話既清楚又有說服力。

Core Idea

cogent 形容論證清楚、有邏輯，而且有足夠說服力把聽者拉向同意。

概念焦點

重點是 convincing clarity：它不是聲量大，而是理由排列得有力量，讓人難以隨便推開。

語氣質地

正式、書面、知性，常用於論證、理由、分析與說明。比 clear 更強，比 persuasive 更重視邏輯結構。

使用提醒

不要把 cogent 用在任何清楚的東西上。說 a cogent UI 不自然；a cogent explanation 或 a cogent argument 才自然。

Meaning

簡短定義

Cogent

表達清楚且有說服力，尤其因為推理本身很扎實。

不是 Coherent

Coherent 是前後一致、結構清楚；cogent 還要能說服人，像理由被壓實後有推力。

Origin

字源來源

cogent 來自 Latin cogere，有 compel、collect、drive together 的意思。這個字保留了理由被推到一起、形成力量的畫面。

想像散落的證據被推到一起，變成一道能推動判斷的力量。這就是 cogent 的感覺。

Memory Hook

記憶鉤子

cogent 是零件彼此扣上後產生推力的論證。

你聽完不只是懂了，還會覺得「這個推論有重量」。那個重量不是權威，而是理由本身排列得夠好。

Usage

生活與專業情境

日常版

她的說明很 cogent，清楚到足以改變整個房間的氣氛。

數位 / 專業版

那份 bug report 很 cogent：它把 regression 指向 timing change，而不是新的 parser。

延伸版

好的 design review 會把 constraints、evidence、trade-offs 與 consequences 串成一條 cogent 的推理鏈。

Collocations

常見搭配與語域

cogent 幾乎總是修飾 argument、case、reason、explanation、analysis 這類與推理有關的名詞。

a cogent argument 正式 / 學術

最典型搭配，表示論點清楚且有說服力。

a cogent explanation 專業 / 教學

表示說明不只清楚，還讓人接受其推理。

a cogent case 法律 / 分析

表示一組理由與證據合在一起，足以支持某主張。

Neighbors

對立與鄰近概念

概念 核心精神 最自然的使用時機
Cogent 清楚、有邏輯且足以說服 談論證、理由、分析、說明、提案、review comment。
Coherent 內部一致、前後連貫、容易理解 當你只想說一段話結構合理，不一定要強調說服力時。

Modern Use

現代延伸

在技術溝通中，cogent 很適合稱讚一段 RCA、proposal 或 review comment：它把證據、因果與取捨串起來。

在評論或法律語境中，cogent 常與 argument、case、evidence 一起出現，表示結論不是只靠情緒，而是被理由支撐。
