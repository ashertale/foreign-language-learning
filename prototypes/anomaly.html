Word 16 / noun

Anomaly

/əˈnɑː.mə.li/

不是普通的「異常」，而是偏離預期模式到值得你停下來追問原因的訊號。

今日定位

把 anomaly 記成「pattern breaker」：它的重點不是怪，而是和正常樣本不一致。

Reading Path 核心 定義 字源 記憶 情境 搭配詞 辨析 現代 來源 回想

Core Idea

anomaly is a meaningful departure from an expected pattern. It asks: why does this case not fit the rest?

概念焦點

一個資料點、事件或行為和已知規律不合，因此需要被辨識、解釋或排除。

語氣質地

正式、分析感強；常見於科學、資料、測試、品質、資安與工程報告。

使用提醒

若只是一般錯誤，用 error；若重點是「偏離正常模式」，用 anomaly。

Meaning

簡短定義

Anomaly

A thing, event, or result that differs from what is normal, expected, or consistent with the surrounding pattern.

不是 Error

error 強調錯；anomaly 先強調不合模式，可能是錯誤，也可能是新現象。

Origin

字源來源

anomaly comes from Greek anomalos, meaning uneven or irregular. The old image is a surface that should be level but suddenly becomes uneven.

字源裡的「uneven」很適合記這個字：不是每個凸起都等於災難，但每個不平整處都值得看一眼。

Memory Hook

記憶鉤子

An anomaly is the one waveform that refuses to line up with the others.

想像一排整齊的訊號，只有一條突然抖了一下。它不一定立刻告訴你答案，但它告訴你：這裡有東西不屬於原本的模式。

Usage

生活與專業情境

日常版

If a shop is usually quiet on Monday but suddenly full, that crowd is an anomaly worth noticing.

數位 / 專業版

In firmware testing, a timing anomaly may appear only under a narrow temperature or power condition.

延伸版

In data analysis, anomaly detection looks for observations that deviate strongly from the expected distribution.

Collocations

常見搭配與語域

Anomaly 常和 detection、data、pattern、result 這類分析字搭配；後面的辨析處理它和 error 的界線。

detect an anomaly data / testing

強調從大量樣本中找出偏離正常行為的事件或資料點。

statistical anomaly research / analytics

指統計上看起來不尋常的結果，可能需要更多資料確認。

anomalous behavior security / systems

形容系統、使用者或設備行為偏離平常模式。

Modern Use

現代延伸

Modern English uses anomaly in science, finance, cybersecurity, manufacturing, and engineering whenever a case does not match the expected model.

The word is useful because it keeps investigation open: an anomaly can reveal a bug, a sensor issue, a rare condition, or a new pattern the model has not learned yet.
