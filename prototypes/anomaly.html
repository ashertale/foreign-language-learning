Word 16 / noun

Anomaly

uh-NAH-muh-lee · UK /əˈnɒm.ə.li/ · US /əˈnɑː.mə.li/

不是普通的「異常」，而是偏離預期模式到值得你停下來追問原因的訊號。

今日定位

把 anomaly 記成「pattern breaker」：它的重點不是怪，而是和正常樣本不一致。

Core Idea

anomaly 是偏離預期模式、因此值得追問原因的異常點。

概念焦點

一個資料點、事件或行為和已知規律不合，因此需要被辨識、解釋或排除。

語氣質地

正式、分析感強；常見於科學、資料、測試、品質、資安與工程報告。

使用提醒

若只是一般錯誤，用 error；若重點是「偏離正常模式」，用 anomaly。

Meaning

簡短定義

Anomaly

和正常、預期或周圍模式不一致的事物、事件或結果。

不是 Error

error 強調錯；anomaly 先強調不合模式，可能是錯誤，也可能是新現象。

Origin

字源來源

anomaly 來自 Greek anomalos，意思是不平、 irregular。字源畫面像原本該平整的表面突然變得不平。

字源裡的「uneven」很適合記這個字：不是每個凸起都等於災難，但每個不平整處都值得看一眼。

Memory Hook

記憶鉤子

anomaly 是那條拒絕和其他訊號對齊的 waveform。

想像一排整齊的訊號，只有一條突然抖了一下。它不一定立刻告訴你答案，但它告訴你：這裡有東西不屬於原本的模式。

Usage

生活與專業情境

日常版

一間店週一通常很安靜，卻突然客滿，這種人潮就是值得注意的 anomaly。

數位 / 專業版

在 firmware testing 中，timing anomaly 可能只在很窄的溫度或電源條件下出現。

延伸版

資料分析中的 anomaly detection 會找出明顯偏離預期分布的觀察值。

Collocations

常見搭配與語域

Anomaly 常和 detection、data、pattern、result 這類分析字搭配；後面的辨析處理它和 error 的界線。

detect an anomaly 資料 / 測試

強調從大量樣本中找出偏離正常行為的事件或資料點。

statistical anomaly 研究 / 分析

指統計上看起來不尋常的結果，可能需要更多資料確認。

anomalous behavior 資安 / 系統

形容系統、使用者或設備行為偏離平常模式。

Neighbors

對立與鄰近概念

概念 核心精神 最自然的使用時機
Anomaly 偏離預期模式，先要求解釋 資料、測試、系統或事件和背景規律不合時
Error 已判定不正確或失敗 當你已經知道結果是錯的，而不是只知道它不尋常

Modern Use

現代延伸

現代英文裡，anomaly 常出現在科學、金融、資安、製造與工程中，描述不符合預期模型的案例。

這個字好用的地方是保留調查空間：anomaly 可能是 bug、sensor issue、罕見條件，也可能是模型尚未學到的新模式。
