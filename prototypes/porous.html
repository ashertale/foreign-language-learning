Porous

POR-us · UK /ˈpɔː.rəs/ · US /ˈpɔːr.əs/

不是鬆散而已，而是本身就有很多可滲透的孔隙。

Core Idea

porous 的好處在於精度：它把情境、語氣和搭配一起推向「本身就有很多可滲透的孔隙」。

概念焦點

判斷點放在 permeable 和 porous 的距離：前者較寬，後者把邊界往「本身就有很多可滲透的孔隙」收。

語氣質地

這個字有一點精修感。自然的關鍵不在正式度，而在上下文是否真的需要「本身就有很多可滲透的孔隙」。

使用提醒

先檢查搭配是否站得住；porous material 自然，不代表任何名詞後面都能硬接 porous。

Meaning

簡短定義

Porous

表示多孔的，也可比喻邊界、系統或規則容易滲透。

不是 Permeable

如果 permeable 已經能讓句子成立，就不必硬換；需要「多孔的，也可比喻邊界、系統或規則容易滲透」時，porous 才有必要。

Usage

生活與專業情境

日常版

porous material 是最低摩擦入口；它讓你不用先背完整定義，也能感到「本身就有很多可滲透的孔隙」。

數位 / 專業版

porous border 適合拿來檢查語域：它讓 porous 從認得走向會判斷。

延伸版

用 porous security 造句時，不要寫得花；只要讓讀者清楚看見「多孔的，也可比喻邊界、系統或規則容易滲透」。

Collocations

Modern Use

現代延伸

實際閱讀時，porous 多半不會孤立出現。先留意 porous material、porous border、porous security 這類搭配如何替它定形。
