Word 18 / noun

Heuristic

/hjʊˈrɪs.tɪk/

不是保證正確的公式，而是在複雜問題裡幫你快速找到可行方向的經驗法則。

今日定位

把 heuristic 記成「useful shortcut for finding」：它犧牲一點完美，換取前進速度。

Reading Path 核心 定義 字源 記憶 情境 搭配詞 辨析 現代 來源 回想

Core Idea

A heuristic is a practical rule, method, or shortcut that helps you discover a workable answer when exhaustive proof is too slow or costly.

概念焦點

用經驗與判斷快速縮小搜尋空間，不承諾絕對正確。

語氣質地

工程、演算法、UX、心理學、決策分析都常見；語氣理性但承認取捨。

使用提醒

不要把 heuristic 當成 algorithm 的同義字；algorithm 更像明確程序，heuristic 更像實用指引。

Meaning

簡短定義

Heuristic

A rule of thumb or practical method that helps solve problems or make decisions, especially when a perfect method is impractical.

不是 Algorithm

algorithm 通常是清楚定義的步驟；heuristic 是幫你快一點接近答案的經驗策略。

too many options
→
apply a rule of thumb
→
find a workable path

Origin

字源來源

heuristic is related to Greek heuriskein, meaning to find or discover. The famous echo is Eureka: the moment of finding.

字源不是「規則」本身，而是「找到」。所以 heuristic 的精神不是法律條文，而是幫你在霧裡找到路標。

Memory Hook

記憶鉤子

A heuristic is a flashlight, not the whole map.

在 debug、設計或決策裡，你常常不能把所有路都走完。heuristic 先照亮最可能有答案的方向，之後還是要用證據驗證。

Usage

生活與專業情境

日常版

A simple heuristic for buying tools is to choose the one you can maintain and actually use, not the one with the longest feature list.

數位 / 專業版

A debugging heuristic is to reproduce the failure first, then change one variable at a time.

延伸版

In AI and search problems, heuristics guide the system toward promising paths without checking every possibility.

Collocations

常見搭配與語域

Heuristic 常和 method、rule、shortcut、bias 搭配；它可以是實用策略，也可能造成偏誤。

a useful heuristic general / professional

指能在真實限制下幫助判斷的經驗法則。

heuristic method engineering / math

指不一定保證最優，但能有效引導求解的方法。

heuristic bias psychology

指人腦捷徑帶來的判斷偏誤，例如過度依賴最近看到的例子。

Neighbors

對立與鄰近概念

概念 核心精神 最自然的使用時機
Heuristic 實用捷徑，幫助探索與決策 問題太大、資訊不完整、需要先找到方向時
Algorithm 明確步驟，通常追求可重複與可證明 當程序已經定義清楚，輸入輸出規則明確時

Modern Use

現代延伸

Modern English uses heuristic in software, AI, human factors, product design, and cognitive psychology.

The word is especially useful when you want to say a method is practical and guiding, but not mathematically guaranteed or universally correct.
