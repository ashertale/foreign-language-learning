Word 78 / noun

Idiosyncrasy

id-ee-oh-SING-kruh-see · UK /ˌɪd.i.əˈsɪŋ.krə.si/ · US /ˌɪd.i.oʊˈsɪŋ.krə.si/

不是大缺點，而是某個人、系統或風格特有、常常重複出現的怪脾氣或獨特小習性。

今日定位

把 idiosyncrasy 記成「a signature quirk that keeps showing up」：不是偶發，而是你熟了就會認出來。

Core Idea

idiosyncrasy 用來描述個體特有、帶點奇特但可辨識的習性、反應方式或風格。

概念焦點

核心是特有性與可辨識，而不是單純奇怪。

語氣質地

常見於人物描寫、系統觀察、設計風格與醫療語境。

使用提醒

不要把 idiosyncrasy 當成 generic quirk；它更像屬於那個個體、反覆出現的 signature 偏差。

Meaning

簡短定義

Idiosyncrasy

不是大缺點，而是某個人、系統或風格特有、常常重複出現的怪脾氣或獨特小習性。

不是 Quirk

quirk 可以是一般怪癖；idiosyncrasy 更強調那是某個人或系統專屬、可辨識的習性。

Origin

字源來源

idiosyncrasy 來自希臘語裡「個人的混合體質」脈絡，後來延伸成一個人或事物獨有的怪癖。

字源畫面可以記成：每個人都像有一套自己的配方，裡面總有別人沒有的那一味。

Memory Hook

記憶鉤子

idiosyncrasy 是你認得出來的那個小偏差。

真正熟悉一個人或系統之後，你通常不是只記住一般規則，而是先記住那些 idiosyncrasies。

Usage

生活與專業情境

日常版

每個人都有一些小小的 idiosyncrasies，像是說話節奏、收納方式或奇怪的偏好。

數位 / 專業版

某些 legacy tools 的 idiosyncrasies 會讓新成員很困惑，但老手反而靠那些特徵快速判斷。

延伸版

人物描寫、精神醫療、系統維護與設計觀察都常用 idiosyncrasy 來談專屬特性。

Collocations

常見搭配與語域

Idiosyncrasy 常和 personal、system、behavioral 這類描述獨特性的詞搭配。

personal idiosyncrasy 人物 / 性格

表示屬於這個人的特有小習性。

system idiosyncrasy 工程 / 維護

表示系統特有的不尋常行為。

behavioral idiosyncrasy 心理 / 觀察

表示可辨識的特殊反應模式。

Neighbors

對立與鄰近概念

概念 核心精神 最自然的使用時機
Idiosyncrasy 專屬於某個個體的可辨識怪癖 談人、風格、工具或 legacy system 時
Quirk 一般的小怪癖或奇特點 不特別強調它是那個個體的 signature 時

Modern Use

現代延伸

現代英文裡，idiosyncrasy 很適合描述那些只有熟悉之後才會開始看見的專屬偏差。

在工程語境中，它常用來談 legacy system 那些文件寫不完、但實際上總會重複出現的特殊習性。
