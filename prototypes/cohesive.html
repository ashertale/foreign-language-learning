Word 59 / adjective

Cohesive

koh-HEE-siv · UK /kəʊˈhiː.sɪv/ · US /koʊˈhiː.sɪv/

不是只是放在一起，而是各部分彼此黏得住、能形成一個完整整體。

今日定位

把 cohesive 記成「parts that truly hold together」：不是排列在同一張桌上，而是真的互相扣住。

Core Idea

cohesive 用來描述團隊、文章、設計或系統的各部分彼此連得起來，不散、不互相打架。

概念焦點

核心是結合成整體，而不是單點都不錯。

語氣質地

常見於寫作、設計、組織與系統描述；比 consistent 更強調整體黏合感。

使用提醒

不要把 cohesive 當成 simply consistent；一致不等於真的有機連在一起。

Meaning

簡短定義

Cohesive

不是只是放在一起，而是各部分彼此黏得住、能形成一個完整整體。

不是 Consistent

consistent 可以只是風格一致；cohesive 更進一步，表示各部分真的扣成一體。

Origin

字源來源

cohesive 和拉丁語裡「黏住、附著」的脈絡有關，所以它的核心一直都不是相像，而是黏合。

字源畫面可以記成：不是把珠子放同一盒，而是真的把它們串成一條線。

Memory Hook

記憶鉤子

cohesive 是每塊都在，但更重要的是每塊都接得上。

一份文件可能每段都寫得不差，可是只有在彼此接得順時，你才會說它 cohesive。

Usage

生活與專業情境

日常版

一套穿搭如果很 cohesive，不是每件都好看而已，而是放在一起很順。

數位 / 專業版

在產品與文件設計裡，cohesive structure 表示功能、文案與視覺不是各做各的。

延伸版

團隊文化、架構設計與研究論文也常用 cohesive 來談整體是否真的連成一氣。

Collocations

常見搭配與語域

Cohesive 常和 strategy、team、design、narrative、structure 搭配。

a cohesive team 組織 / 團隊

表示人不只是聚在一起，而是真的能協同。

a cohesive narrative 寫作 / 研究

表示段落與論點彼此扣緊。

cohesive design system 設計 / 產品

表示元件和語彙能形成整體經驗。

Neighbors

對立與鄰近概念

概念 核心精神 最自然的使用時機
Cohesive 各部分真的連成一體 談文件、團隊、設計、架構或敘事時
Consistent 風格或行為保持一致 只想說穩定一致，不特別強調整體黏合時

Modern Use

現代延伸

現代英文裡，cohesive 常被拿來評價文章、產品與團隊，因為很多問題不是單點差，而是整體接不起來。

在工程與設計語境中，它特別適合描述一個系統是否由內到外都在講同一套邏輯。
