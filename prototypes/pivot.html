Pivot

PIV-uht · UK /ˈpɪv.ət/ · US /ˈpɪv.ət/

不是把之前全丟掉，而是抓住支點，沿著那個支點換方向。

今日定位

把 pivot 記成「turn around a useful axis」：不是亂轉，而是圍著關鍵支點換路線。

Core Idea

pivot 用來描述人在策略、產品、身體動作或論點上做出方向調整，同時保留部分核心。

概念焦點

核心是帶著支點轉向，不是重來。

語氣質地

常見於商務、產品、體育、寫作與現場應變。

使用提醒

不要把 pivot 當成 abandon；pivot 常表示某些核心保留，方向重新設定。

Meaning

簡短定義

Pivot

不是把之前全丟掉，而是抓住支點，沿著那個支點換方向。

不是 Change

change 很廣；pivot 更像圍著一個關鍵點轉向，而不是全部打掉重練。

Origin

字源來源

pivot 原本就是旋轉時承重的那個點，所以它後來被借去描述策略或身體如何轉向非常自然。

字源畫面可以記成：不是把桌子抬起重擺，而是抓住中心點讓它順勢轉過去。

Memory Hook

記憶鉤子

pivot 是不浪費原有力量地轉向。

好的 pivot 不是 panic flip，而是先看清楚哪個核心值得保留，再換方向。

Usage

生活與專業情境

日常版

聊天時如果話題走偏，你可以自然地 pivot back to what actually matters.

數位 / 專業版

產品團隊決定 pivot from feature breadth to workflow depth，不代表前面全白做，而是重心改了。

延伸版

商業模式、產品策略、體育動作與即時溝通都常用 pivot。

Collocations

常見搭配與語域

Pivot 常和 strategy、toward、back、from-to 搭配。

pivot the strategy 商務 / 產品

表示策略沿著支點轉向。

pivot toward reliability 工程 / 組織

表示重心往新方向移。

pivot back to the point 對話 / 寫作

表示把討論拉回主線。

Neighbors

對立與鄰近概念

概念 核心精神 最自然的使用時機
Pivot 圍著核心支點重新轉向 談產品、策略、對話或現場應變時
Change 一般地發生改變 只想說有變，不強調支點與轉向結構時

Modern Use

現代延伸

現代英文裡，pivot 已經是產品與商務語境裡的高頻字，因為好調整很少是完全重來。

在工程討論中，它也適合描述在失敗中換方向，但保留真正有價值的核心。
