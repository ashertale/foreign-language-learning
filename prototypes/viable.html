Word 85 / adjective

Viable

VY-uh-buhl · UK /ˈvaɪ.ə.bəl/ · US /ˈvaɪ.ə.bəl/

不是完美，而是能活、能撐、能作為真正可用的選項存在。

今日定位

把 viable 記成「able to live or hold up in the real world」：重點不是好看，而是能活下去。

Core Idea

viable 用來描述方案、產品、商業模式或生物體具備存活、運作或持續成立的能力。

概念焦點

核心是可持續存在，不只是勉強能做。

語氣質地

常見於商務、產品、策略、生物與研究討論。

使用提醒

不要把 viable 當成等於 ideal；一個 viable option 只是可活，不保證最漂亮。

Meaning

簡短定義

Viable

形容方案、產品、商業模式或生物體具備存活、運作或持續成立的能力。

不是 Feasible

feasible 側重做不做得出來；viable 更看做出來之後能不能繼續活、繼續成立。

先確認能做
→
再看能不能撐住
→
最後成為真正可用選項

Origin

字源來源

viable 和拉丁語裡「生命」的脈絡相關，所以它天生帶著能活、能存續的味道。

字源畫面可以記成：不是種子剛發芽而已，而是它有機會長成能活下去的植株。

Memory Hook

記憶鉤子

viable 是它不只做得出來，還活得下去。

很多方案 technical feasible，但未必 long-term viable。這兩者常常不是同一件事。

Usage

生活與專業情境

日常版

搬到新城市的計畫也許可行，但是否 viable 還要看工作、生活與支持系統能不能撐。

數位 / 專業版

一個 prototype 也許能跑，但只有在維護、成本與風險都扛得住時，它才算 viable.

延伸版

商業模式、產品策略、生物學與醫療都常用 viable 來談存續能力。

Collocations

常見搭配與語域

Viable 常和 option、model、solution、cells 搭配。

a viable option 決策 / 策略

表示它是能真正採用的選項。

commercially viable 商務 / 產品

表示商業上能活下去。

viable solution 工程 / 設計

表示方案不只成立，還撐得住。

Neighbors

對立與鄰近概念

概念 核心精神 最自然的使用時機
Viable 能活、能撐、能持續成立 談商業模式、產品方案或長期選項時
Feasible 條件下做得出來 只想驗證可執行，不討論後續存續時

Modern Use

現代延伸

現代英文裡，viable 很常用來提醒團隊：能做出來和做得長久，是兩個不同層次。

在工程與產品決策裡，它能幫你把討論從 demo success 拉到長期維護與成本現實。
