Extraneous

ek-STRAY-nee-us · UK /ɪkˈstreɪ.ni.əs/ · US /ɪkˈstreɪ.ni.əs/

不是多一點而已，而是多到和核心無關。

今日定位

先不要把 extraneous 翻成一個固定中文；先看上下文是不是正在要求你判斷「多到和核心無關」。

Core Idea

用 extraneous 時，句子通常在做一個收焦動作：把讀者帶到「多到和核心無關」那個更窄的位置。

概念焦點

如果上下文沒有讓「多到和核心無關」變重要，這個字就不必上場。

語氣質地

它的語氣像把句子往內收一格：少一點泛稱，多一點可判斷的輪廓。

使用提醒

使用時要避開翻譯腔：先有情境，再讓 extraneous 進來，不要先決定一定要用它。

Meaning

簡短定義

Extraneous

表示外來的、無關的、多餘的。

不是 Irrelevant

irrelevant 可以先說出 不相關；extraneous 則把句子推到「多到和核心無關」這個更細的位置。

Usage

生活與專業情境

日常版

先拿 extraneous detail 寫一個你真的會遇到的場景；這比多看三個中文解釋有用。

數位 / 專業版

讀 extraneous variable 時，注意它前後通常會提供理由，讓「多到和核心無關」變得必要。

延伸版

extraneous noise 是延伸練習。把它放進句子後，確認意思仍回到「多到和核心無關」。

Collocations

常見搭配與語域

把搭配當成三個入口：extraneous detail 給基本場景，extraneous variable 看語域，extraneous noise 測你能不能輸出。

extraneous detail 寫作 / 編輯

extraneous detail 先建立基本語境；它讓定義有地方落腳。

extraneous variable 研究 / 分析

用 extraneous variable 練習時，先看它前後是否支持「多到和核心無關」。

extraneous noise 工程 / 訊號

Neighbors

對立與鄰近概念

概念 核心精神 最自然的使用時機
Extraneous 外來的、無關的、多餘的 當句子需要把「多到和核心無關」說清楚，而不是只留在 不相關 的外圈時。
Irrelevant 不相關 當你只需要先說 不相關，不必把語氣收進「多到和核心無關」時。

Modern Use

現代延伸

遇到 extraneous 時，先看旁邊的名詞或動詞。它們會告訴你是 寫作 / 編輯、研究 / 分析，還是 工程 / 訊號 的用法。

真正掌握它的標準是：你能換一個情境，仍然說清楚為什麼這裡該用 extraneous。
